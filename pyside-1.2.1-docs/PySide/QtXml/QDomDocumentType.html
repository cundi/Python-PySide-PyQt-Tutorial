Detailed Description¶

The PySide.QtXml.QDomDocumentType class is the representation of the DTD in the document tree.

The PySide.QtXml.QDomDocumentType class allows read-only access to some of the data structures in the DTD: it can return a map of all PySide.QtXml.QDomDocumentType.entities() and PySide.QtXml.QDomDocumentType.notations() . In addition the function PySide.QtXml.QDomDocumentType.name() returns the name of the document type as specified in the &lt;!DOCTYPE name&gt; tag. This class also provides the PySide.QtXml.QDomDocumentType.publicId() , PySide.QtXml.QDomDocumentType.systemId() and PySide.QtXml.QDomDocumentType.internalSubset() functions.
