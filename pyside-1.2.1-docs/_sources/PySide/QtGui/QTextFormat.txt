Detailed Description
--------------------



    The :class:`PySide.QtGui.QTextFormat` class provides formatting information for a :class:`PySide.QtGui.QTextDocument` .

    A :class:`PySide.QtGui.QTextFormat` is a generic class used for describing the format of parts of a :class:`PySide.QtGui.QTextDocument` . The derived classes :class:`PySide.QtGui.QTextCharFormat` , :class:`PySide.QtGui.QTextBlockFormat` , :class:`PySide.QtGui.QTextListFormat` , and :class:`PySide.QtGui.QTextTableFormat` are usually more useful, and describe the formatting that is applied to specific parts of the document.

    A format has a ``FormatType`` which specifies the kinds of text item it can format; e.g. a block of text, a list, a table, etc. A format also has various properties (some specific to particular format types), as described by the Property enum. Every property has a corresponding Property.

    The format type is given by :meth:`PySide.QtGui.QTextFormat.type` , and the format can be tested with :meth:`PySide.QtGui.QTextFormat.isCharFormat` , :meth:`PySide.QtGui.QTextFormat.isBlockFormat` , :meth:`PySide.QtGui.QTextFormat.isListFormat` , :meth:`PySide.QtGui.QTextFormat.isTableFormat` , :meth:`PySide.QtGui.QTextFormat.isFrameFormat` , and :meth:`PySide.QtGui.QTextFormat.isImageFormat` . If the type is determined, it can be retrieved with :meth:`PySide.QtGui.QTextFormat.toCharFormat` , :meth:`PySide.QtGui.QTextFormat.toBlockFormat` , :meth:`PySide.QtGui.QTextFormat.toListFormat` , :meth:`PySide.QtGui.QTextFormat.toTableFormat` , :meth:`PySide.QtGui.QTextFormat.toFrameFormat` , and :meth:`PySide.QtGui.QTextFormat.toImageFormat` .

    A format's properties can be set with the :meth:`PySide.QtGui.QTextFormat.setProperty` functions, and retrieved with :meth:`PySide.QtGui.QTextFormat.boolProperty` , :meth:`PySide.QtGui.QTextFormat.intProperty` , :meth:`PySide.QtGui.QTextFormat.doubleProperty` , and :meth:`PySide.QtGui.QTextFormat.stringProperty` as appropriate. All the property IDs used in the format can be retrieved with allPropertyIds(). One format can be merged into another using :meth:`PySide.QtGui.QTextFormat.merge` .

    A format's object index can be set with :meth:`PySide.QtGui.QTextFormat.setObjectIndex` , and retrieved with :meth:`PySide.QtGui.QTextFormat.objectIndex` . These methods can be used to associate the format with a :class:`PySide.QtGui.QTextObject` . It is used to represent lists, frames, and tables inside the document.
